# Orchestra-Core

Marketing/website front-end for Orchestra-Core — a private, local-first AI financial literacy coach.

See `CLAUDE.md` for full project context (brand, design system, page structure, roadmap).

## Tech stack

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Getting started

Requires Node.js & npm.

```sh
# Install dependencies
npm i

# Start the dev server
npm run dev

# Build for production
npm run build
```
